# Справочник по синтаксису Markdown

Этот гайд поможет тебе быстро оформить любой файл, например, README.md. Подробности можно найти в руководстве GitHub.

## 1. Заголовки
Количество решеток означает уровень заголовка (от 1 до 6).

```markdown
# Заголовок 1 (Название проекта)
## Заголовок 2 (Разделы: Установка, Использование)
### Заголовок 3 (Подразделы)
```

## 2. Оформление текста

```markdown
**Жирный текст** (для акцентов)
*Курсив* (для пояснений)
~~Зачеркнутый текст~~
```

## 3. Списки

```markdown
* Элемент маркированного списка
* Еще один элемент

1. Первый пункт нумерованного списка
2. Второй пункт
```

## 4. Код и блоки
Внутри строки используются одинарные бэктики: `command`.
Блок кода с подсветкой синтаксиса:

```python
def hello_world():
    print("Hello, GitHub!")
```
